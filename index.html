Export / Import Config

Export: Generate a setup code that includes the API token and all user accounts. Share it with users to configure the app on a new device.

Import: Paste a setup code received from an admin to configure this device with all users and token.

Smartsheet API Token

The API token is stored securely as a server environment variable (SMARTSHEET_TOKEN). To change it, update the variable in Vercel → Project → Settings → Environment Variables and redeploy.
